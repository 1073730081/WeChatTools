接口帮助
接口地址：

http://vip.weixin139.com/weixin/wx_domain.php?user=您的用户名&key=您的KEY&domain=域名

查询日志：

http://www.weixin139.com/weixin/log?user=您的用户名

接口返回参数：

{"from":"node.js","status":0,"errmsg":"域名正常","user":"xxx","domain":"www.baidu.com","endtime":"2038-01-19"}

名称 释义 说明
domain 域名 被检测的域名，不带http://，不支持URL
user 用户名 接口用户名
status 检测结果 0为正常，1为被转码，2为被封，3查询失败，-1接口到期。注意：频率过高默认返回3状态
errmsg 提示 中文提示信息
endtime 接口有效期 该接口的有效截止日期时间
